## 二、前后端字段不匹配

### `/api/simulate` 返回格式错误

**后端 `api.py:270-294` 返回**：
```python
{
  "current": {
    "bits": "...",           # ❌ 前端期望 "code"
    "hexagram": "...",       # ❌ 前端期望 "name"
    "physics_desc": "...",    # ❌ 前端期望 "physics_description"
    "conf_m1": 0.85,         # ⚠️ 前端FSMNode无此字段
  },
  "flips": [...]
}
```

**前端 `useStore.ts` 期望**：
```typescript
current: FSMNode { code, name, physics_description, entropy_S, mass_M }
```

**影响**：`simulateFlip()` 调用时 `data.current.hexagram` 为 `undefined`

2. **后端 `/api/simulate` 字段名错误**：需统一为 `code`/`name`/`physics_description`
3. **前端无置信度仪表盘组件**：需新增 Canvas/SVG 绘制的迈速表 UI
4. **无打字机效果组件**：需新增 Typewriter effect
5. **无概率分布柱状图组件**：需新增 Chart.js 或自绘
